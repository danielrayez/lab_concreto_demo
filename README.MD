## MCP (Model Context Protocol)

Este repositorio incluye `.vscode/mcp.json` con un servidor MCP local de sistema de archivos.

**Requisito:** Node.js instalado (para ejecutar `npx`).

> **Nota sobre el error 403 de Power BI MCP:**
> El endpoint `https://api.fabric.microsoft.com/v1/mcp/powerbi` requiere una licencia de
> Microsoft Fabric habilitada en el tenant. Si tu tenant no la tiene, obtendrás el error
> `"The feature is not available"`. La configuración local incluida en este repo no depende
> de ese servicio.
